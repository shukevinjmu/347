Blog post #5: JavaScript to load HTML

Today's blog post is focused on loading HTML with javascript. The reason I wanted to do this is because I've been manually updating my blog's format(navbar/links) for each blog post which is tedious.
While you briefly showcased this in one of your videos, I thought it would a good exercise to try for myself.

While googling, I saw that one issue with doing this happens when users block JavaScript. I imagine for robustness, it would be good to not do this for important content.

One method I found was using jQuery: 

                
$(document).ready(function(){
   
    $('#content').load("new.html");
                     
});

This appears to be the shortest(in code length) way which makes sense given that its jQuery.
For the sake of the post though, I want to search for the pure JavaScript solution.

I found my answer in this stackoverflow post.
I then tried to replace my navbar with this: 

                
fetch('navbar.html')
  .then(data => data.text())
  .then(html => document.getElementById('navbar-load').innerHTML = html);

The only issue I had with this is that it broke my JavaScript DOM references for the side panel. This meant that when I clicked the button, the file did not work.
